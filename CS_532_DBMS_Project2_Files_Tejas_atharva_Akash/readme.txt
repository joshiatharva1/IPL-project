You can follow the below instructions to run the Web application :

1) All the four queries and the code for plotting the Graphical data analysis is written in one Python file called mainQuery.py

2) The index.html webpage is placed in a folder called templates and it should stay as it is

3) Open cmd on the location where mainQuery.py file is located and type "python mainQuery.py" to compile and execute the code

4) A webserver will start running and the address to open the webpage will be displayed in the cmd. Example of cmd given below 

C:\Users\tejas\OneDrive\Documents\F23_CS_532_DBMS\Proj2_NoSQL>python mainQuery.py

WARNING: This is a development server. Do not use it in a production deployment. Use a production WSGI server instead.
 * Running on http://127.0.0.1:5000
Press CTRL+C to quit
 * Restarting with stat
 * Debugger is active!
 * Debugger PIN: 530-240-782

5) Click on the link which is displayed besides Running on .. in this case it is http://127.0.0.1:5000 and you will be directed to the landing page where you can see all of the 4 tasks
